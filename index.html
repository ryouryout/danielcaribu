USB / WebHID
DualShock 4 / DualSense / Edge / VR2 Sense

接続から保存までを順に進める校正画面

USB 接続した PlayStation コントローラーの校正と確認を行います。 接続、中心調整、可動域調整、確認、保存の順に進めてください。

USB 接続のみ Chrome / Edge HTTPS 公開URL

開始前の確認

  • 公開URLを PC の Chrome / Edge で開く
  • USB ケーブルで接続する
  • バッテリー残量を十分に確保する
  • 処理中はブラウザを閉じない

保存前の確認

  • 中心校正と可動域校正を先に終える
  • 必要ならクイックテストと表示値で確認する
  • 問題がある場合は保存せず再調整する
  • 永続保存は最後に一度だけ行う

1. 起動

公開URLから開く

GitHub Pages の HTTPS URL を使ってください。ファイルを直接開く方法では動作しません。

2. 接続

USB 接続して認証する

接続ボタンを押し、対象のコントローラーを 1 台だけ選択してください。

3. 校正

確認してから保存する

中心調整、可動域調整、必要なら微調整を行い、最後に本体へ保存してください。

操作手順

接続から保存までの手順

  1. USB で接続し、「接続」を押します。
  2. 中心校正またはクイック校正を行います。
  3. 可動域校正を行い、必要なら微調整を開きます。
  4. 表示値やクイックテストで結果を確認します。
  5. 問題がなければ本体へ保存します。

補足

Bluetooth 接続では処理できません。接続に失敗する場合は USB 接続へ切り替えてください。

前回接続

未保存の変更が残っている可能性があります。必要なら再接続後に保存してください。

   このブラウザでは利用できません。

スマートフォンやタブレットは対象外です。
